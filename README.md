## 🎯 一句话说明

**A股探子** 是一个聚焦"龙头股"的 A股分析微信小程序，帮助用户：
- 📊 快速浏览**概念板块**异动
- 🐉 一键查看**成分股详情**（K线/资金/公告）
- ⭐ 维护**自选股**，实时查看行情
- 📰 浏览**热点事件**，关联相关个股
