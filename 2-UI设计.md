## 1. 信息架构

### 页面结构
```
首页（市场概览）
├── 概念板块列表（按热度排序）
│   └── 板块详情（龙头股列表）
├── 热点事件
│   └── 事件详情（关联标的）
├── 自选股管理
└── 设置（推送频率、数据源等）
```

### 核心交互流程
1. **首页** → 今日热门概念板块 Top 10 → 点击进入详情
2. **事件流** → 最新时事 → 自动匹配标的 → 加入自选
3. **自选** → 关注的股票列表 → 实时行情监控

## 2. 视觉风格

### 配色方案（金融专业风格）
| 用途 | 颜色 | Hex |
|------|-----|-----|
| 涨 | 红色 | #E53935 |
| 跌 | 绿色 | #43A047 |
| 中性/平盘 | 灰色 | #9E9E9E |
| 主色（背景） | 深蓝黑 | #1A1A2E |
| 次色（卡片） | 深灰蓝 | #16213E |
| 强调色 | 金色 | #FFD700 |
| 文字主色 | 白色 | #FFFFFF |
| 文字次色 | 浅灰 | #B0B0B0 |

### 字体规范
- 标题：18px, Bold
- 副标题：16px, SemiBold
- 正文：14px, Regular
- 辅助文字：12px, Regular

### 图标
- 使用 SF Symbols 或等效开源图标库
- 涨：↑ 红色箭头
- 跌：↓ 绿色箭头
- 板块：□ 方形图标
- 自选：★ 星标

## 4. 技术实现建议

### 微信小程序框架
- 使用 Taro 或 uni-app（跨平台）
- 状态管理：MobX 或 Vuex

### 实时推送方案
- WebSocket 长连接
- 行情数据：东方财富/同花顺 WebSocket API
- 轮询备选：每 30 秒刷新

### 数据存储
- 自选股：本地缓存 + 云端同步
- 用户偏好：本地存储
